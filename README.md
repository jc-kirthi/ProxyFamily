# 👵 Proxy Family
**World's First Blockchain-Verified Relative Call Deflection System.**

*"We show up so you don't have to."*

---

## 🚀 The Problem
Indian relatives. The constant calls about your marriage, your salary, and why your cousin in America is doing better than you. Scope creep exists in families too. 

## 🛡️ The Solution: Proxy Family
Proxy Family uses **Voice Cloning Simulation**, **Sentiment Analysis (Passive Aggression Meter)**, and **Algorand Blockchain** to deflect unwanted calls with immutable, AI-generated excuses.

### 🌟 Core Features (Hackathon Ready)
1. **Interactive Call Deflection**: A full WhatsApp-style calling interface that handles the "Annoying Relative" interaction.
2. **AI Voice Clone (Mock)**: Uses Web Speech API to simulate your voice sounds, trained on 10,000 hours of Indian family drama.
3. **Sentiment Analysis**: Real-time tracking of "Passive Aggression" and "Guilt Tripping" levels.
4. **Blockchain-Verified Excuses**: Your lies are now immutable! Every deflection creates a proof on the Algorand Testnet.
5. **Saree Mode**: A respectful UI layer for when you're feeling cultured. Changes "Deflect" to "Respectfully Decline".
6. **Gas Meter (₹)**: Because even lying has a cost (mock gas fees shown in Rupees).

## 🛠️ System Architecture
- **Frontend**: React, TailwindCSS, Framer Motion, Lucide Icons.
- **AI**: Gemini 2.0 Flash (Annoying Relative Persona) + Web Speech API (Deflections).
- **Blockchain**: Algorand (Mock/Testnet Anchoring).
- **Resilience**: Robust Offline/Fallback services ensure the demo works even if the venue Wi-Fi dies.

## ⚡ Demo Skit Flow
1. **The Ring**: Relative (Judge) calls.
2. **The Passive Aggression**: App plays an AI-generated annoying relative response.
3. **The Shutdown**: User clicks "Deflect". Proxy Family clones voice, delivers excuse, and anchors it on the blockchain.
4. **The Proof**: Show the "Blockchain Verified" hash and the incrementing "Aunties Deflected" counter.
